## How to use the program?
- Download the [`.exe`](https://github.com/Mrmisterxd/white-screen/releases/tag/v1.0) file from the repository, double-click the file, and you can use the script!
- The script is used like a standard [`.py`](https://github.com/Mrmisterxd/white-screen/blob/main/Whitescrenn.py) file.
- The .exe is simply a compiled .py file created using pyinstaller.

## How can this program be useful?
- This program can help students who are studying and do not have a desk lamp.
- This program will highlight textbooks/any other material.
### *Basically, this program displays a white screen on the entire screen, thereby highlighting everything on the table with white light.*

## How do I exit this program?
- To exit this program you need to press the esc key
- Or the easiest way is to press Alt+F4 or close the program with Alt+Tab
